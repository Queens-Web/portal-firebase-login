# portal-firebase-login

## Usage

Fill in config details in src > firebase.jsx <br/>
Config details come from <br/>
FireBase/Project name: Portal/Project settings/General <br/>
Under "Your apps". Select CDN.<br/>

```
apiKey: "",
authDomain: "",
databaseURL: "",
projectId: "",
storageBucket: "",
messagingSenderId: "",
appId: "",
measurementId: ""

```

npm start
